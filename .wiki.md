# Project Summary
The project is a web-based hidden object game titled **聊斋奇谭** (Liao Zhai Qi Tan), which immerses players in traditional Chinese folklore through interactive storytelling and engaging gameplay. Players navigate captivating levels filled with hidden items and challenges, providing both entertainment and education. Recent enhancements include the integration of levels 4, 5, and 6, featuring new backgrounds and item sprites, as well as improved user interface elements and gameplay mechanics. The game now includes a quantity selector for purchasing hints and knives, enhancing the purchasing experience. Additionally, the project has undergone a significant update to improve asset management by converting large PNG files to JPG format for better performance.

# Project Module Description
The project consists of several functional modules:
- **Frontend**: Built with React, it manages game interactions, navigation, and rendering of game scenes.
- **Game Logic**: Manages game states, including player progress and level completion.
- **Routing**: Facilitates navigation between pages such as the main menu and level selection.
- **Audio Management**: Oversees background music and sound effects for an immersive experience.
- **Inventory Management**: Manages player items and displays them in a backpack dialog.
- **Purchase Management**: Handles item purchasing, including the new quantity selector for hints and knives.

# Usage
1. **Install Dependencies**:
   Run the following command to install the necessary packages:
   ```bash
   pnpm install
   ```

2. **Build the Project**:
   To build the project for production, use:
   ```bash
   pnpm run build
   ```

3. **Run the Development Server**:
   Start the development server with:
   ```bash
   pnpm run dev
   ```

4. **Lint the Code**:
   To check for code quality issues, run:
   ```bash
   pnpm run lint
   ```
